## 베스트셀러 분석을 통한 독서 트렌드 탐색 및 도서 추천 시스템
| 2025-2학기 텍스트마이닝 강의에서 팀 프로젝트로 진행

### 프로젝트 개요
팬데믹을 기점으로 사회 전반의 독서 관심도가 증가하였습니다.
본 프로젝트는 "우리는 왜 다시 책을 펼치게 되었나?" 라는 질문에서 출발하여,
베스트셀러 데이터와 독자 리뷰 텍스트를 분석해 근 5년간의 독서 트렌드를 탐색합니다.

#### 연구 목적
1. 2020~2024년 베스트셀러를 분석하여 근 5년간의 독서 트렌드 흐름 파악
2. 사회적 이슈와 도서 트렌드 간의 연관성 분석
3. 독자 리뷰 텍스트의 감성 분석을 통해 사회 정서의 변화를 정량적으로 해석

### 데이터
데이터 소스
- 출처: 교보문고, 알라딘 웹사이트
- 기간: 2020~2024년 (최근 5년간 연간 베스트셀러)

### 분석 프레임워크
#### 01. 데이터 수집
- 교보문고, 알라딘 연도별 베스트셀러 목록 활용
- Selenium 등을 활용한 리뷰 데이터 크롤링

#### 02. 전처리
- 메타데이터: 중복 제거, 결측값 처리, 연도 및 장르 표준화
- 텍스트 데이터: 한글 정규화, 불용어 제거, 토큰화, 품사 태깅
- 분석용 변수 생성: 토큰 수, 감성 점수 등

#### 03. 분석
- 장르 및 출판 트렌드 분석: 연도별 장르 분포 변화
- 키워드 빈도 분석: 연도별 관심 주제 파악
- 감성 분석: 도서 리뷰 기반 감성 점수 지표 생성
- 토픽 모델링: 잠재 주제 추출

#### 04. 시각화
- matplotlib, seaborn — 트렌드 시각화
- wordcloud — 연도별 대표 키워드 시각화
- Sankey Diagram / Heatmap — 토픽 흐름 및 변화 시각화

#### 05. 인사이트 도출
- 연도별 토픽·키워드·감성 점수 변화로 관심사 흐름의 전환 파악
- 특정 시기 주요 사회적 이슈와 독서 트렌드 간의 연결성 탐색
- 감성 점수의 증감과 주제 전환을 바탕으로 연도별 독서 분위기 해석

### 기대 효과
#### 학술적 가치
- 감성 지표를 통해 독자 정서를 정량적으로 분석하여 출판 시장과 사회문화 맥락의 연관성 파악
- 시기별 독서 분위기 및 관심사 연구의 기반 마련

#### 산업적 활용
- 감성 지표를 활용한 맞춤형 출판 기획 및 마케팅 전략 수립
- 독자 정서 흐름을 기반으로 트렌드 대응형 콘텐츠 개발

#### 확장 가능성
- 영화, 공연, 뉴스 등 타 문화 콘텐츠 영역으로 확장 가능
- 시계열 기반 트렌드 예측 및 사회 정서 흐름 지표화
